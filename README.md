## Backend (Golang)

### 📁 Requirements

- Go 1.18+
- Parquet files inside `backend/data/` (e.g., `sample1.parquet`, `sample2.parquet`, etc.)

### ▶️ Run Backend

```bash
cd backend
go mod tidy
go run main.go
```

## Frontend (React+Vite)

### ▶️ Run Backend
```bash
cd frontend
npm install
npm run dev
```
